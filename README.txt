GLOBAL COVID VISUALIZER

TECHNOLOGIES
This project uses React, MaterialUI, ReactGlobe, ReactBootstrap, and Victory.

DATA PROCESSING
Before starting the project, the data is processed using python (pycountry lib) 
to get the coordinates of the countries in the covid_dataset data.

PROCEDURE
In order to run this web app locally in from your browser, you will need to do
the following things:

1. Type 'npm i' to download all the dependencies.
2. Type 'npm start' to run the program on your browser.
